# React vs Next.js: Full Comparison

| Feature | React | Next.js |
|---------|-------|---------|
| Routing | Manual (React Router) | File-based (automatic) |
| Data Fetching | Client-side only | Server + Client |
| Rendering | CSR only | SSR, SSG, ISR, CSR |
| API Backend | Separate server needed | Built-in API routes |
| Image Optimization | Manual | Automatic |
| SEO | Poor (empty HTML) | Excellent (pre-rendered) |
| Code Splitting | Manual setup | Automatic |
| Deployment | Complex | Zero-config (Vercel) |
